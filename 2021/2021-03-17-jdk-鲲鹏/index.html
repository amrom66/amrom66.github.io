jdk-鲲鹏

作者： linjinbao66 时间： March 17, 2021 分类：

安装版本：1.8.0_191

安装方式：磁盘安装

参考链接： http://www.cloud-onlinelab.cn/

  1. 下载编译好的openjdk包

    wget wget https://obs-mirror-ftp4.obs.cn-north-4.myhuaweicloud.com/systemreliable/OpenJDK8U-jdk_aarch64_linux_hotspot_8u191b12.tar.gz

  2. 解压

    tar -zxvf OpenJDK8U-jdk_aarch64_linux_hotspot_8u191b12.tar.gz
mv jdk8u191-b12 /opt/

  3. 配置环境变量

    /etc/profile

    export PATH=$PATH:$MYSQL_HOME/bin
export JAVA_HOME=/opt/jdk
export PATH=$PATH:${JAVA_HOME}/bin

    source /etc/profile

  4. 验证

    [root@dbmaster ~]# java -version
openjdk version "1.8.0_191"
OpenJDK Runtime Environment (AdoptOpenJDK)(build 1.8.0_191-b12)
OpenJDK 64-Bit Server VM (AdoptOpenJDK)(build 25.191-b12, mixed mode)
[root@dbmaster ~]#
