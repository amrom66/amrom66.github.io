es-鲲鹏

作者： linjinbao66 时间： March 17, 2021 分类：

安装版本：6.4.2

安装方式：磁盘安装

  1. 下载

    wget https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-6.4.2.tar.gz

  2. 解压

    tar -zxvf elasticsearch-6.4.2.tar.gz 
mkdir /usr/local/es
mv elasticsearch-6.4.2 elasticsearch
cp elasticsearch /usr/local/es/

  3. 修改配置

    /usr/local/es/elasticsearch/config/elasticsearch.yml

    cluster.name: elasticsearch
node.name: node-1
path.data: /usr/local/es/elasticsearch/data
path.logs: /usr/local/es/elasticsearch/logs
network.host: 0.0.0.0
http.port: 9200
xpack.ml.enabled: false
bootstrap.system_call_filter: false

  4. 调整系统参数

    vim /etc/security/limits.conf

    * soft nofile 65536
* hard nofile 65536

    vim /etc/sysctl.conf

    vm.max_map_count=655360

    sysctl -p

  5. 增加elastic用户

    useradd elastic
chown -R elastic:elastic /usr/local/es/elasticsearch/
chmod -R 755 /usr/local/es

  6. 启动与验证

    sh /usr/local/es/elasticsearch/bin/elasticsearch
