redis-鲲鹏

作者： linjinbao66 时间： March 17, 2021 分类：

安装版本：4.0.3

部署方式：单机部署

安装方式：磁盘安装

  1. 下载组件依赖

    yum install -y jemalloc jemalloc-devel

  2. 下载安装包

    wget https://obs-mirror-ftp4.obs.cn-north-4.myhuaweicloud.com:443/database/redis-4.0.3-aarch64.tar.gz

  3. 解压安装

    tar -xvf  redis-4.0.3-aarch64.tar.gz 
mv redis-4.0.3 /opt/
mv /opt/redis-4.0.3 /opt/redis

  4. 修改redis.conf配置（可选）

  5. 启动

    cd /opt/redis/bin
redis-server

  6. 客户端连接验证

    [root@dbmaster bin]# ./redis-cli -h 127.0.0.1 -p 6379
127.0.0.1:6379> set name lin1
OK
127.0.0.1:6379> get name
"lin1"
127.0.0.1:6379> exit
