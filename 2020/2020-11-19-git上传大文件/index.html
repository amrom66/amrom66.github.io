git上传大文件

作者： linjinbao 时间： November 19, 2020 分类：

github默认文件限制阿紫50M内，超出50M会报警告，超出100M无法上传。如果仍然想上传，可以使用git large file system，上限1GB。

brew install git-lfs
git lfs install
git rm --cached bin/bin/kubelet. ##清除已经加入本地的记录
git lfs track "bin/kubelet"
cat .gitattributes #上述命令会在.gitattributes文件中追加记录
git add .gitattrubutes ##git 官方文档强烈建议加入
git add bin/kubelet
git commit -m "update"
git push
