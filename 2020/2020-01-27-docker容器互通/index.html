docker 容器互通

作者： linjinbao 时间： January 27, 2020 分类：

docker容器互通

创建docker网络

docker create network -d bridge --ip-range=192.168.1.0/24 --gateway=192.168.1.1 --subnet=192.168.1.0/24 bridge2

创建容器

docker run -dit --network=bridge2 --ip=192.168.1.4 busgbox
docker run -dit --network=bridge2 --ip=192.168.1.3 busgbox

此时两个容器可以互通，使用ip a命令可以看到docker创建的两个网卡
