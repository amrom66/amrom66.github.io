rpm工具用法

作者： linjinbao66 时间： November 4, 2019 分类：

rpm工具用法

  1. rpm -ivh * 安装软件包
  2. rpm -Uvh 升级软件包
  3. rpm -e 卸载软件包
  4. rpm -qa 查看系统已经安装的rpm包
  5. rpm -q 后面接包名，查询指定的包是否安装
  6. rpm -qi 查询已安装的包的信息
  7. rpm -ql 查询软件包安装了哪些文件
  8. rpm -qf 查询一个文件是否那个包安装的

rpm安装traceroute

  1. 下载rpm包并上传到服务器 traceroute-2.0.22-2.el7.x86_64.rpm
  2. 执行命令 rpm -ivh tracrroute_**_.rpm
  3. traceroute ip
