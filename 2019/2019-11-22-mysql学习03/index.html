第3章-mysql系统文件

  1. 查询慢查询设置

    ```sql

    show variables like ‘%long%';

    show variables like ‘log_slow_queries’;

    set long_query_time=1; #1秒

show variables like ‘slow_query_log’; # 日志是否开启 show variables like ‘slow_query_log_file’; # 日志位置

show variables like ‘log_queries_not_using_indexes’; # 以索引判断

慢查询的sql记录存放在`mysql.slow_slow`中
