两数之和

作者： linjinbao66 时间： April 6, 2022 分类：

题目：两数之和 II - 输入有序数组

给你一个下标从 1 开始的整数数组 numbers ，该数组已按 非递减顺序排列  ，请你从数组中找出满足相加之和等于目标数 target 的两个数。如果设这两个数分别是 numbers[index1] 和 numbers[index2] ，则 1 <= index1 < index2 <= numbers.length 。

以长度为 2 的整数数组 [index1, index2] 的形式返回这两个整数的下标 index1 和 index2。

你可以假设每个输入 只对应唯一的答案 ，而且你 不可以 重复使用相同的元素。

你所设计的解决方案必须只使用常量级的额外空间。

解答：

class Solution {
    public int[] twoSum(int[] numbers, int target) {
        int length = numbers.length;
        int left = 0;
        int right = length-1;
        if(numbers[left]>=0){
            for(int i=length-1; i>0; i--){
                if(numbers[i] > target) continue;
                int tmp = target - numbers[i];
                for(int j=0; j<i; j++){
                    if(tmp == numbers[j]) return new int[]{j+1, i+1};
                }
            }
        }else if(numbers[right] <= 0){
            for(int i=0; i<length; i++){
                if(numbers[i] < target) continue;
                int tmp = numbers[i] - target;
                for(int j=length-1; j>i; j--){
                    if(tmp == numbers[j]) return new int[]{i+1, j+1};
                }
            }
        }
        int index = 0;
        for(int i=0; i<length; i++){
            if(numbers[i] > 0) {
                index = i;
                break;
            }
        }
        int begin = 0;
        if(numbers[index] < target) begin = index;
        for(int i=begin; i<length; i++){
            int tmp = target - numbers[i];
            for(int j=begin+1; j<length; j++){
                if(tmp == numbers[j]) return new int[]{i+1, j+1};
            }
        }
        
        return null;
    }
}
