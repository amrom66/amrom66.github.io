两个数组的交集

作者： linjinbao66 时间： April 7, 2022 分类： leetcode

题目

给你两个整数数组 nums1 和 nums2 ，请你以数组形式返回两数组的交集。返回结果中每个元素出现的次数，应与元素在两个数组中都出现的次数一致（如果出现次数不一致，则考虑取较小值）。可以不考虑输出结果的顺序。

示例 1：

输入：nums1 = [1,2,2,1], nums2 = [2,2] 输出：[2,2] 示例 2:

输入：nums1 = [4,9,5], nums2 = [9,4,9,8,4] 输出：[4,9]

解答

class Solution {
    public int[] intersect(int[] nums1, int[] nums2) {
        int index1 = 0;
        int index2 = 0;
        int[] nums = new int[Math.min(nums1.length, nums2.length)];
        int index = 0;
        Arrays.sort(nums1);
        Arrays.sort(nums2);
        boolean tag = false; 
        while(index1 < nums1.length && index2 < nums2.length){
            if(nums1[index1] == nums2[index2]){
                nums[index] = nums1[index1];
                index ++;
                index1 ++;
                index2 ++;
                tag = true;
            }else if(nums1[index1] < nums2[index2]){
               index1 ++;
               tag = false;
            }else {
                index2 ++;
                tag = false;
            }
        }
        if(!tag && index < nums.length && index2 < nums2.length){
            for(int i=index2; i<nums2.length; i++){
                if(nums1[index1-1] == nums2[i]){
                    nums[index] = nums1[index1-1];
                    index ++;
                    break;
                }
            }
        }
        if(!tag && index < nums.length && index1 < nums1.length){
            for(int i=index1; i<nums1.length; i++){
                if(nums1[i] == nums2[index2-1]){
                    nums[index] = nums1[index2-1];
                    index ++;
                    break;
                }
            }
        }
        int[] tmp = new int[index];
        for(int i=0; i<index; i++){
            tmp[i] = nums[i];
        }  
        return tmp;
    }
}
